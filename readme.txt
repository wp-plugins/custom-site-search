Install a Google Custom Site Search widget for your wordpress blog.
== Description ==
The search application allows visitors to search your website or the web from within your site/blog. This means that the user does not have to leave your page or even open a new tab/window. It uses google custom site search to search within your website/blog and shows the result in a bottom docked panel without making the site unresponsive. When the user clicks on any link in the search result the page is opened in new tab.

== Installation ==

Follow the procedure to install Custom Site Search for you blog/site.

1. Download custom-site-search.zip
2. Unzip custom-site-search.zip and upload `BumpIn_Site_Search.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag Drop the Custom Site Search widget to your sidebar according to your need.(goto *Presentation-> widgets*)
5. Customize and Register your widget(FREE!) by visiting socialbar.BumpIn.com (If you have not already registered)
6. Paste the code you got from BumpIn website in the text box and click the "Save Changes" button
7. Verify your email address used for registering at socialbar.BumpIn.com and set your password by clicking on the link sent to your email account
8. Re-customize your widget by logging in at socialbar.BumpIn.com using your email address as your login name
9. If you re-customize the widget from the SocialBar admin panel, the changes would automatically be reflected on your site/blog
10. If you forgot your password you can reset it from http://socialbar.bumpin.com/site/forgot_password/ by verifying your email id
